# Yepl MVC UI

Starter React application built with Vite, TypeScript, React Router, Material UI, Jest, and React Testing Library.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run test
npm run test:watch
```

## Stack

- Vite for development and bundling
- React 19 with TypeScript
- React Router for navigation
- Material UI for the application shell and starter views
- Jest with jsdom and React Testing Library for component testing

## Structure

- src/components contains shared UI shells
- src/pages contains route-level pages
- src/theme contains the Material UI theme
- src/test contains test setup and helpers

## Notes

The app starts with a routed layout, a home page, a docs page, and a not-found route so feature work has a stable structure from the start.
